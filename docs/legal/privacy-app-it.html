Riepilogo

OpenClient non raccoglie, memorizza, trasmette né condivide alcun dato personale con lo sviluppatore. L'app funziona interamente sul tuo dispositivo e comunica solo con il server che configuri. Non vengono inviati analisi, telemetria o report di crash allo sviluppatore.

Nessun server dello sviluppatore

OpenClient non dispone di server, backend, servizi di analisi o infrastrutture cloud gestiti dallo sviluppatore. Non esistono telemetria, tracciamento dell'utilizzo o report di crash inviati allo sviluppatore. Lo sviluppatore non riceve mai alcun dato dall'utilizzo dell'app.

Archiviazione locale e iCloud

Tutti i dati dell'app vengono archiviati sul tuo dispositivo o nel tuo account iCloud personale. Lo sviluppatore non ha accesso a nessuno di essi. Nello specifico:

  • Cronologia delle conversazioni — archiviata localmente nella directory dei documenti dell'app. Se iCloud Drive è attivo, le conversazioni vengono sincronizzate tra i tuoi dispositivi Apple tramite il tuo spazio di archiviazione iCloud Drive personale.
  • Impostazioni e profilo utente — archiviati localmente in UserDefaults. Se iCloud è attivo, i dati del profilo vengono sincronizzati tramite iCloud Key-Value Store, privato per il tuo Apple ID.
  • Elementi di memoria — archiviati localmente in UserDefaults. Se iCloud è attivo, gli elementi di memoria vengono sincronizzati tramite iCloud Key-Value Store, privato per il tuo Apple ID. Ciò include gli elementi creati automaticamente dai modelli durante le conversazioni quando vengono invocati gli strumenti di memoria dell'agente.
  • Chiavi API e URL del server — archiviate esclusivamente nel Portachiavi del dispositivo tramite il framework Security di iOS/macOS. Non vengono mai trasmesse allo sviluppatore.
  • Indice Spotlight — i titoli delle conversazioni e gli estratti dei messaggi vengono indicizzati localmente in Spotlight per consentire la ricerca sul dispositivo. Questi dati non lasciano il tuo dispositivo.

Nulla lascia il tuo dispositivo, ad eccezione delle richieste che invii esplicitamente al server che configuri.

Comunicazione con il server

L'app si connette esclusivamente all'URL del server che fornisci — un'istanza Ollama, un proxy LiteLLM o qualsiasi altro backend compatibile con OpenAI. Tutte le richieste vengono inviate direttamente dal tuo dispositivo al tuo server. Ciò include:

  • Messaggi di chat e contesto della conversazione
  • Immagini e documenti allegati ai messaggi (Vision)
  • Dati audio per la trascrizione vocale (Speech-to-Text tramite /v1/audio/transcriptions)
  • Testo per la sintesi vocale (Text-to-Speech tramite /v1/audio/speech)
  • Prompt per la generazione di immagini (/v1/images/generations)
  • Query di ricerca web, che il tuo server LiteLLM invia al provider di ricerca che hai configurato lato server (es.: Brave, Tavily, DuckDuckGo). L'app non contatta mai direttamente i provider di ricerca.
  • Input e output delle chiamate agli strumenti quando la modalità agente è attiva

Lo sviluppatore non ha accesso, conoscenza né coinvolgimento in questa comunicazione.

Contenuto condiviso e importato

Il contenuto ricevuto tramite l'estensione Condivisione, lo schema URL personalizzato (openclient://), il trascinamento o i Comandi rapidi Apple viene elaborato interamente sul tuo dispositivo e inviato solo al server configurato. Lo sviluppatore non riceve né ha accesso ad alcun contenuto condiviso.

Nessun SDK di terze parti né analisi

OpenClient ha zero dipendenze esterne per la sua funzionalità in fase di esecuzione. Non sono integrati nell'app reti pubblicitarie, framework di tracciamento né SDK di analisi. L'unico strumento di terze parti utilizzato in fase di compilazione è SwiftLint, uno strumento di analisi statica del codice che viene eseguito durante la compilazione e non è mai incluso nel binario dell'app pubblicata.

L'app integra l'SDK Votice — un progetto open source dello stesso autore — esclusivamente per consentire agli utenti di inviare segnalazioni di bug e richieste di funzionalità dalla schermata Impostazioni. Le segnalazioni sono completamente anonime: nessuna identità utente, account, identificatore del dispositivo né dati personali vengono allegati a una segnalazione. Quando utilizzi questa funzionalità, solo il testo che inserisci volontariamente viene inviato al servizio Votice. Nessun altro dato dell'app (conversazioni, chiavi API, URL del server, memoria o allegati) viene mai inviato a Votice. L'utilizzo di questa funzionalità è completamente facoltativo.

Open source e verificabile

OpenClient è completamente open source con licenza Apache 2.0. Puoi ispezionare, controllare e verificare ogni riga di codice nel repository GitHub. Questa trasparenza ti consente di confermare che non esiste alcuna raccolta di dati.

Privacy dei minori

L'app non raccoglie consapevolmente informazioni da nessuno, inclusi i minori di 13 anni. Poiché non vengono raccolti dati, non esiste alcun rischio di raccogliere dati di minori.
