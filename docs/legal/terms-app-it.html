Accettazione

Scaricando, installando o utilizzando OpenClient («l'App»), accetti questi termini. Se non sei d'accordo, non utilizzare l'App.

Cosa fa l'app

OpenClient è un client Apple nativo per iOS, iPadOS e macOS che si connette a qualsiasi server compatibile con OpenAI che fornisci — come Ollama, LiteLLM o qualsiasi altro backend self-hosted o di terze parti che implementa l'API OpenAI. L'App consente conversazioni di testo, analisi di immagini e documenti (Vision), trascrizione vocale (Speech-to-Text), riproduzione testo-parlato, generazione di immagini, ricerca web e un ciclo agentico di chiamata agli strumenti — tutto elaborato esclusivamente dal server che configuri. L'App non ospita, gestisce né controlla alcun server LLM, modello AI o provider di ricerca.

Le funzionalità aggiuntive includono:

  • Sincronizzazione delle conversazioni: Le conversazioni vengono archiviate localmente sul tuo dispositivo e, se iCloud è attivo, sincronizzate tra i tuoi dispositivi Apple tramite iCloud Drive.
  • Memoria: Gli elementi di memoria persistenti vengono archiviati in iCloud Key-Value Store (quando disponibile) o localmente in UserDefaults, e iniettati nelle conversazioni come contesto. I modelli possono anche creare elementi di memoria in modo autonomo durante una conversazione.
  • Ricerca web: Quando attivata, l'App invia query di ricerca tramite il tuo server LiteLLM al provider di ricerca configurato su quel server (es.: Brave, Tavily, DuckDuckGo). L'App non contatta mai direttamente i provider di ricerca.
  • Modalità agente: L'App supporta un ciclo agentico di chiamata agli strumenti in cui il modello può invocare strumenti registrati e ricevere i loro output prima di generare una risposta finale.
  • Estensione Condivisione e Comandi rapidi: L'App può ricevere contenuti da altre app tramite l'estensione Condivisione di iOS/iPadOS, uno schema URL personalizzato (openclient://), il trascinamento o i Comandi rapidi Apple.
  • Spotlight: I titoli delle conversazioni e gli estratti dei messaggi vengono indicizzati localmente in Spotlight per la ricerca rapida. Questi dati non lasciano il tuo dispositivo.

App gratuita, costi API di terze parti

OpenClient è gratuita da scaricare e utilizzare. L'App stessa non ha abbonamenti, acquisti in-app né costi nascosti di alcun tipo.

Tuttavia, collegare l'App a un provider LLM a pagamento — come OpenAI, Anthropic, Groq o qualsiasi altro servizio cloud — potrebbe comportare addebiti fatturati direttamente da quel provider in base all'utilizzo dell'API (token consumati, immagini generate, audio trascritto, ecc.). Questi costi sono completamente indipendenti dall'App e lo sviluppatore di OpenClient non ha visibilità, controllo né responsabilità su di essi.

I modelli locali e self-hosted (es.: Ollama sul proprio hardware) generalmente non comportano costi per richiesta, ma sei responsabile dell'infrastruttura che gestisci.

Dati locali e iCloud

L'App archivia la cronologia delle conversazioni localmente sul tuo dispositivo usando il file system. Se iCloud è attivo nelle impostazioni del dispositivo, i seguenti dati potrebbero essere sincronizzati con iCloud:

  • File delle conversazioni (tramite iCloud Drive)
  • Impostazioni del profilo utente (tramite iCloud Key-Value Store)
  • Elementi di memoria (tramite iCloud Key-Value Store)

Controlli se la sincronizzazione iCloud è attiva tramite le impostazioni iCloud del tuo dispositivo. Lo sviluppatore non accede ad alcun dato archiviato nel tuo account iCloud. Disattivare la sincronizzazione iCloud significa che i dati rimangono solo localmente e non vengono condivisi tra i dispositivi.

Queste cifre sono solo approssimazioni; gli addebiti effettivi sono determinati esclusivamente dal tuo provider LLM o piano di utilizzo API. Lo sviluppatore non è responsabile di eventuali discrepanze tra le stime visualizzate e la fatturazione effettiva.
