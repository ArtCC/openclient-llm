Αποδοχή

Κατεβάζοντας, εγκαθιστώντας ή χρησιμοποιώντας το OpenClient («η Εφαρμογή»), αποδέχεστε αυτούς τους όρους. Εάν δεν συμφωνείτε, μην χρησιμοποιείτε την Εφαρμογή.

Τι κάνει η εφαρμογή

Το OpenClient είναι ένας εγγενής Apple client για iOS, iPadOS και macOS που συνδέεται με οποιονδήποτε διακομιστή συμβατό με OpenAI παρέχετε — όπως Ollama, LiteLLM ή οποιοδήποτε άλλο αυτο-φιλοξενούμενο ή τρίτο backend που υλοποιεί το OpenAI API. Η Εφαρμογή επιτρέπει συνομιλίες κειμένου, ανάλυση εικόνων και εγγράφων (Vision), μεταγραφή ομιλίας (Speech-to-Text), αναπαραγωγή κειμένου σε ομιλία, δημιουργία εικόνων, web αναζήτηση και έναν αγεντικό βρόχο κλήσης εργαλείων — όλα επεξεργαζόμενα αποκλειστικά από τον διακομιστή που διαμορφώνετε. Η Εφαρμογή δεν φιλοξενεί, δεν λειτουργεί ούτε ελέγχει κανέναν διακομιστή LLM, μοντέλο AI ή πάροχο αναζήτησης.

Οι πρόσθετες δυνατότητες περιλαμβάνουν:

  • Συγχρονισμός συνομιλιών: Οι συνομιλίες αποθηκεύονται τοπικά στη συσκευή σας και, εάν το iCloud είναι ενεργό, συγχρονίζονται μεταξύ των συσκευών Apple σας μέσω του iCloud Drive.
  • Μνήμη: Τα μόνιμα στοιχεία μνήμης αποθηκεύονται στο iCloud Key-Value Store (όταν είναι διαθέσιμο) ή τοπικά στο UserDefaults, και εισάγονται στις συνομιλίες ως συμφραζόμενα. Τα μοντέλα μπορούν επίσης να δημιουργούν αυτόνομα στοιχεία μνήμης κατά τη διάρκεια μιας συνομιλίας.
  • Web αναζήτηση: Όταν είναι ενεργοποιημένη, η Εφαρμογή αποστέλλει ερωτήματα αναζήτησης μέσω του διακομιστή LiteLLM σας στον πάροχο αναζήτησης που έχει διαμορφωθεί σε εκείνον τον διακομιστή (π.χ. Brave, Tavily, DuckDuckGo). Η Εφαρμογή δεν επικοινωνεί ποτέ απευθείας με παρόχους αναζήτησης.
  • Λειτουργία πράκτορα: Η Εφαρμογή υποστηρίζει έναν αγεντικό βρόχο κλήσης εργαλείων στον οποίο το μοντέλο μπορεί να καλεί καταχωρημένα εργαλεία και να λαμβάνει τις εξόδους τους πριν δημιουργήσει μια τελική απάντηση.
  • Επέκταση Κοινής χρήσης και Συντομεύσεις: Η Εφαρμογή μπορεί να λαμβάνει περιεχόμενο από άλλες εφαρμογές μέσω της επέκτασης Κοινής χρήσης iOS/iPadOS, ενός προσαρμοσμένου σχήματος URL (openclient://), μεταφοράς και απόθεσης, ή Συντομεύσεων Apple.
  • Spotlight: Οι τίτλοι συνομιλιών και τα αποσπάσματα μηνυμάτων ευρετηριάζονται τοπικά στο Spotlight για γρήγορη αναζήτηση. Αυτά τα δεδομένα δεν φεύγουν από τη συσκευή σας.

Δωρεάν εφαρμογή, κόστος API τρίτων

Το OpenClient είναι δωρεάν για λήψη και χρήση. Η ίδια η Εφαρμογή δεν έχει συνδρομές, αγορές εντός εφαρμογής ούτε κρυφές χρεώσεις οποιουδήποτε είδους.

Ωστόσο, η σύνδεση της Εφαρμογής με έναν πληρωμένο πάροχο LLM — όπως OpenAI, Anthropic, Groq ή οποιαδήποτε άλλη υπηρεσία cloud — μπορεί να προκαλέσει χρεώσεις που τιμολογούνται απευθείας από εκείνον τον πάροχο βάσει της χρήσης του API (tokens που καταναλώθηκαν, εικόνες που δημιουργήθηκαν, ήχος που μεταγράφηκε κ.λπ.). Αυτά τα κόστη είναι πλήρως ανεξάρτητα από την Εφαρμογή και ο προγραμματιστής του OpenClient δεν έχει ορατότητα, έλεγχο ή ευθύνη γι' αυτά.

Τα τοπικά και αυτο-φιλοξενούμενα μοντέλα (π.χ. Ollama στο δικό σας hardware) συνήθως δεν συνεπάγονται κόστος ανά αίτημα, αλλά είστε υπεύθυνοι για την υποδομή που λειτουργείτε.

Οι ευθύνες σας

  • Διακομιστής και υποδομή: Είστε υπεύθυνοι για τη διαμόρφωση, συντήρηση και ασφάλεια του δικού σας διακομιστή και τυχόν ανάντη παρόχων LLM με τους οποίους συνδέεται.
  • Κλειδιά API και κόστος: Οποιαδήποτε κλειδιά API διαμορφώσετε στην Εφαρμογή ή στον διακομιστή σας είναι δική σας ευθύνη. Τα κόστη που προκύπτουν μέσω παρόχων LLM (OpenAI, Anthropic, Groq, Ollama κ.λπ.) είναι αποκλειστικά δική σας ευθύνη ως χειριστής. Ο προγραμματιστής του OpenClient δεν έχει εμπλοκή ούτε ευθύνη για χρεώσεις τρίτων υπηρεσιών.
  • Περιεχόμενο: Είστε ο μοναδικός υπεύθυνος για το περιεχόμενο που αποστέλλετε μέσω της Εφαρμογής και τις απαντήσεις που λαμβάνετε. Ο προγραμματιστής δεν παρακολουθεί, δεν ελέγχει ούτε αποθηκεύει συνομιλίες.
  • Περιεχόμενο web αναζήτησης: Είστε υπεύθυνοι για τα ερωτήματα που αποστέλλετε μέσω της λειτουργίας web αναζήτησης και για τυχόν ενέργειες που πραγματοποιείτε βάσει των αποτελεσμάτων. Τα αποτελέσματα παρέχονται από τρίτες υπηρεσίες μέσω του διακομιστή LiteLLM σας· η ακρίβεια, διαθεσιμότητα και νομική συμμόρφωσή τους δεν εμπίπτουν στον έλεγχο του προγραμματιστή.
  • Μνήμη: Είστε υπεύθυνοι για την αναθεώρηση και διαχείριση των στοιχείων μνήμης που αποθηκεύει η Εφαρμογή, συμπεριλαμβανομένων αυτών που δημιουργήθηκαν αυτόνομα από μοντέλα. Ελέγχετε τακτικά τη μνήμη σας στις Ρυθμίσεις για να διασφαλίσετε ότι το αποθηκευμένο περιεχόμενο ανταποκρίνεται στις προσδοκίες σας.
  • Κοινόχρηστο και εισαγόμενο περιεχόμενο: Είστε υπεύθυνοι για οποιοδήποτε περιεχόμενο αποστέλλεται στην Εφαρμογή μέσω της επέκτασης Κοινής χρήσης, του σχήματος URL, της μεταφοράς και απόθεσης, ή των Συντομεύσεων Apple.
  • Συμμόρφωση: Συμφωνείτε να χρησιμοποιείτε την Εφαρμογή σύμφωνα με όλους τους εφαρμοστέους νόμους και τους όρους χρήσης οποιουδήποτε παρόχου LLM ή αναζήτησης με τον οποίο συνδέεστε.

Τοπικά δεδομένα και iCloud

Η Εφαρμογή αποθηκεύει το ιστορικό συνομιλιών τοπικά στη συσκευή σας χρησιμοποιώντας το σύστημα αρχείων. Εάν το iCloud είναι ενεργό στις ρυθμίσεις της συσκευής σας, τα ακόλουθα δεδομένα ενδέχεται να συγχρονιστούν με το iCloud:

  • Αρχεία συνομιλιών (μέσω iCloud Drive)
  • Ρυθμίσεις προφίλ χρήστη (μέσω iCloud Key-Value Store)
  • Στοιχεία μνήμης (μέσω iCloud Key-Value Store)

Ελέγχετε εάν ο συγχρονισμός iCloud είναι ενεργός μέσω των ρυθμίσεων iCloud της συσκευής σας. Ο προγραμματιστής δεν έχει πρόσβαση σε δεδομένα που αποθηκεύονται στον λογαριασμό iCloud σας. Η απενεργοποίηση του συγχρονισμού iCloud σημαίνει ότι τα δεδομένα παραμένουν μόνο τοπικά και δεν κοινοποιούνται μεταξύ συσκευών.

Εκτιμώμενο κόστος

Η Εφαρμογή ενδέχεται να εμφανίζει εκτιμώμενο κόστος συνομιλίας που υπολογίζεται από τη χρήση tokens και τα μεταδεδομένα τιμολόγησης που παρέχει ο διακομιστής LiteLLM σας. Αυτά τα στοιχεία είναι μόνο προσεγγίσεις· οι πραγματικές χρεώσεις καθορίζονται αποκλειστικά από τον πάροχο LLM ή το πρόγραμμα χρήσης API σας. Ο προγραμματιστής δεν ευθύνεται για τυχόν αποκλίσεις μεταξύ των εμφανιζόμενων εκτιμήσεων και της πραγματικής τιμολόγησης.

Άδεια ανοιχτού κώδικα

Ο πηγαίος κώδικας της Εφαρμογής είναι διαθέσιμος υπό την άδεια Apache 2.0. Μπορείτε να επιθεωρήσετε, τροποποιήσετε και αναδιανείμετε τον κώδικα υπό τους όρους εκείνης της άδειας. Η φύση ανοιχτού κώδικα του project σάς επιτρέπει να επαληθεύσετε όλους τους ισχυρισμούς που γίνονται σε αυτούς τους όρους και στην Πολιτική απορρήτου.

Χωρίς εγγύηση

Η Εφαρμογή παρέχεται «ως έχει» χωρίς καμία εγγύηση οποιουδήποτε είδους, ρητή ή σιωπηρή, συμπεριλαμβανομένων, χωρίς περιορισμό, εγγυήσεων εμπορευσιμότητας, καταλληλότητας για συγκεκριμένο σκοπό ή μη παραβίασης. Ο προγραμματιστής δεν εγγυάται ότι η Εφαρμογή θα είναι χωρίς σφάλματα, αδιάλειπτη ή συμβατή με όλες τις διαμορφώσεις διακομιστή.

Περιορισμός ευθύνης

Στο μέγιστο βαθμό που επιτρέπεται από την εφαρμοστέα νομοθεσία, ο προγραμματιστής δεν θα ευθύνεται για έμμεσες, παρεπόμενες, ειδικές, αποθετικές ή κυρωτικές ζημίες, ή οποιαδήποτε απώλεια δεδομένων, εσόδων ή κερδών, που προκύπτουν από τη χρήση της Εφαρμογής. Αυτό περιλαμβάνει, χωρίς περιορισμό, κόστη που προκύπτουν μέσω παρόχων API τρίτων, απώλεια δεδομένων που προκύπτει από εσφαλμένη διαμόρφωση διακομιστή, ανακριβή αποτελέσματα αναζήτησης ή ακούσιες ενέργειες που εκτελούνται από μοντέλα σε λειτουργία πράκτορα.

Χωρίς συνεργασία

Το OpenClient είναι ένα ανεξάρτητο project ανοιχτού κώδικα. Δεν σχετίζεται, εγκρίνεται ούτε χρηματοδοτείται από τις LiteLLM, OpenAI, Anthropic, Ollama, Groq, Brave, Tavily, Apple ή οποιαδήποτε άλλη εταιρεία ή υπηρεσία που αναφέρεται στην Εφαρμογή ή στην τεκμηρίωσή της.

Αλλαγές σε αυτούς τους όρους

Αυτοί οι όροι ενδέχεται να ενημερώνονται κατά καιρούς. Οι αλλαγές θα δημοσιεύονται σε αυτή τη σελίδα με ενημερωμένη ημερομηνία αναθεώρησης. Η συνεχόμενη χρήση της Εφαρμογής μετά από αλλαγές συνιστά αποδοχή των νέων όρων.
