Περίληψη

Το OpenClient δεν συλλέγει, αποθηκεύει, μεταδίδει ούτε κοινοποιεί προσωπικά δεδομένα στον προγραμματιστή. Η εφαρμογή λειτουργεί εξ ολοκλήρου στη συσκευή σας και επικοινωνεί μόνο με τον διακομιστή που διαμορφώνετε. Δεν αποστέλλονται αναλυτικά στοιχεία, τηλεμετρία ή αναφορές σφαλμάτων στον προγραμματιστή.

Χωρίς διακομιστή προγραμματιστή

Το OpenClient δεν διαθέτει διακομιστή, backend, υπηρεσία ανάλυσης ή υποδομή cloud που λειτουργεί από τον προγραμματιστή. Δεν υπάρχει τηλεμετρία, παρακολούθηση χρήσης ή αναφορές σφαλμάτων που αποστέλλονται στον προγραμματιστή. Ο προγραμματιστής δεν λαμβάνει ποτέ δεδομένα από τη χρήση σας της εφαρμογής.

Τοπική αποθήκευση και iCloud

Όλα τα δεδομένα της εφαρμογής αποθηκεύονται στη συσκευή σας ή στον προσωπικό σας λογαριασμό iCloud. Ο προγραμματιστής δεν έχει πρόσβαση σε κανένα από αυτά. Συγκεκριμένα:

  • Ιστορικό συνομιλιών — αποθηκεύεται τοπικά στον κατάλογο εγγράφων της εφαρμογής. Εάν το iCloud Drive είναι ενεργό, οι συνομιλίες συγχρονίζονται μεταξύ των συσκευών Apple σας μέσω του προσωπικού σας χώρου αποθήκευσης iCloud Drive.
  • Ρυθμίσεις και προφίλ χρήστη — αποθηκεύονται τοπικά στο UserDefaults. Εάν το iCloud είναι ενεργό, τα δεδομένα προφίλ συγχρονίζονται μέσω του iCloud Key-Value Store, ιδιωτικό για το Apple ID σας.
  • Στοιχεία μνήμης — αποθηκεύονται τοπικά στο UserDefaults. Εάν το iCloud είναι ενεργό, τα στοιχεία μνήμης συγχρονίζονται μέσω του iCloud Key-Value Store, ιδιωτικό για το Apple ID σας. Αυτό περιλαμβάνει στοιχεία που δημιουργήθηκαν αυτόματα από μοντέλα κατά τη διάρκεια συνομιλιών όταν καλούνται τα εργαλεία μνήμης του πράκτορα.
  • Κλειδιά API και URL διακομιστή — αποθηκεύονται αποκλειστικά στο Keychain της συσκευής μέσω του Security framework του iOS/macOS. Δεν μεταδίδονται ποτέ στον προγραμματιστή.
  • Ευρετήριο Spotlight — οι τίτλοι συνομιλιών και αποσπάσματα μηνυμάτων ευρετηριάζονται τοπικά στο Spotlight για αναζήτηση στη συσκευή. Αυτά τα δεδομένα δεν φεύγουν από τη συσκευή σας.

Τίποτα δεν φεύγει από τη συσκευή σας, εκτός από τα αιτήματα που στέλνετε ρητά στον διακομιστή που διαμορφώνετε.

Επικοινωνία με τον διακομιστή

Η εφαρμογή συνδέεται αποκλειστικά με το URL του διακομιστή που παρέχετε — μια παρουσία Ollama, ένα proxy LiteLLM ή οποιοδήποτε άλλο backend συμβατό με OpenAI. Όλα τα αιτήματα αποστέλλονται απευθείας από τη συσκευή σας στον διακομιστή σας. Αυτό περιλαμβάνει:

  • Μηνύματα chat και συμφραζόμενα συνομιλίας
  • Εικόνες και έγγραφα που επισυνάπτονται σε μηνύματα (Vision)
  • Δεδομένα ήχου για μεταγραφή ομιλίας (Speech-to-Text μέσω /v1/audio/transcriptions)
  • Κείμενο για σύνθεση ομιλίας (Text-to-Speech μέσω /v1/audio/speech)
  • Prompts για δημιουργία εικόνων (/v1/images/generations)
  • Ερωτήματα web αναζήτησης, τα οποία ο διακομιστής LiteLLM σας προωθεί στον πάροχο αναζήτησης που έχετε διαμορφώσει στον διακομιστή (π.χ. Brave, Tavily, DuckDuckGo). Η εφαρμογή δεν επικοινωνεί ποτέ απευθείας με παρόχους αναζήτησης.
  • Είσοδοι και έξοδοι κλήσεων εργαλείων όταν η λειτουργία πράκτορα είναι ενεργή

Ο προγραμματιστής δεν έχει πρόσβαση, γνώση ούτε εμπλοκή σε αυτή την επικοινωνία.

Κοινόχρηστο και εισαγόμενο περιεχόμενο

Το περιεχόμενο που λαμβάνεται μέσω της επέκτασης Κοινής χρήσης, του προσαρμοσμένου σχήματος URL (openclient://), της μεταφοράς και απόθεσης, ή των Συντομεύσεων Apple επεξεργάζεται εξ ολοκλήρου στη συσκευή σας και αποστέλλεται μόνο στον διαμορφωμένο διακομιστή. Ο προγραμματιστής δεν λαμβάνει ούτε έχει πρόσβαση σε κοινόχρηστο περιεχόμενο.

Χωρίς SDK τρίτων ή αναλύσεις

Το OpenClient έχει μηδέν εξωτερικές εξαρτήσεις για τη λειτουργικότητα χρόνου εκτέλεσης. Δεν υπάρχουν ενσωματωμένα δίκτυα διαφημίσεων, frameworks παρακολούθησης ούτε SDK αναλύσεων στην εφαρμογή. Το μοναδικό εργαλείο τρίτων που χρησιμοποιείται κατά τη μεταγλώττιση είναι το SwiftLint, ένα εργαλείο στατικής ανάλυσης κώδικα που εκτελείται κατά τη δημιουργία και δεν περιλαμβάνεται ποτέ στο εκτελέσιμο αρχείο της δημοσιευμένης εφαρμογής.

Η εφαρμογή ενσωματώνει το Votice SDK — ένα project ανοιχτού κώδικα του ίδιου συγγραφέα — αποκλειστικά για να επιτρέπει στους χρήστες να υποβάλλουν αναφορές σφαλμάτων και αιτήματα λειτουργιών από την οθόνη Ρυθμίσεων. Οι αναφορές είναι πλήρως ανώνυμες: δεν επισυνάπτεται ταυτότητα χρήστη, λογαριασμός, αναγνωριστικό συσκευής ούτε προσωπικά δεδομένα σε μια υποβολή. Όταν χρησιμοποιείτε αυτή τη λειτουργία, μόνο το κείμενο που εισάγετε εθελοντικά αποστέλλεται στην υπηρεσία Votice. Κανένα άλλο δεδομένο εφαρμογής (συνομιλίες, κλειδιά API, URL διακομιστή, μνήμη ή συνημμένα) δεν αποστέλλεται ποτέ στο Votice. Η χρήση αυτής της λειτουργίας είναι εντελώς προαιρετική.

Ανοιχτός κώδικας και ελέγξιμο

Το OpenClient είναι πλήρως ανοιχτού κώδικα υπό την άδεια Apache 2.0. Μπορείτε να επιθεωρήσετε, ελέγξετε και επαληθεύσετε κάθε γραμμή κώδικα στο αποθετήριο GitHub. Αυτή η διαφάνεια σάς επιτρέπει να επιβεβαιώσετε ότι δεν υπάρχει συλλογή δεδομένων.

Απόρρητο παιδιών

Η εφαρμογή δεν συλλέγει σκόπιμα πληροφορίες από κανέναν, συμπεριλαμβανομένων παιδιών κάτω των 13 ετών. Δεδομένου ότι δεν συλλέγονται δεδομένα, δεν υπάρχει κίνδυνος συλλογής δεδομένων παιδιών.

Αλλαγές σε αυτή την πολιτική

Εάν αυτή η πολιτική ενημερωθεί, οι αλλαγές θα δημοσιευτούν σε αυτή τη σελίδα με ενημερωμένη ημερομηνία αναθεώρησης. Δεδομένου ότι η εφαρμογή δεν συλλέγει δεδομένα, σημαντικές αλλαγές είναι απίθανες.
