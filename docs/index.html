OpenClient LLM

A native Apple client for LiteLLM. Connect to any LLM provider through a single unified endpoint with a beautiful SwiftUI experience. Run local models or use privacy-respecting APIs — your data, your server, your rules.

💬

Real-time Streaming

Chat with any LLM using Server-Sent Events for instant, token-by-token responses.

🪟

Liquid Glass Design

Built with Apple's Liquid Glass design language for a native, modern feel.

🤖

Any LLM Provider

Ollama, OpenAI, Anthropic, Groq, and more — all through your LiteLLM server.

📱

Multi-platform

Native experience on iPhone, iPad, and Mac from a single shared codebase.

🔒

Privacy First

Self-hosted — your data stays on your server. API keys stored securely in Keychain.

🌍

Multilingual

Available in English, Spanish, French, Italian, German, Portuguese, Japanese, and more.
