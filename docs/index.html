FEATURES

Everything you need to talk to any LLM

🤖

Any LLM Provider

Connect directly to Ollama, LiteLLM, or any OpenAI-compatible server — Anthropic, Groq, Gemini, and more.

🔒

Privacy First

Self-hosted — your data stays on your server. API keys stored securely in Keychain.

💬

Real-time Streaming

Chat with any LLM using Server-Sent Events for instant, token-by-token responses.

🔍

Web Browsing

Search the web and get up-to-date answers — the model browses in real time to enrich its responses.

🖼️

Image Generation

Generate images from text prompts via DALL·E, Stable Diffusion, Gemini, and more.

🎙️

Voice & Audio

Speech-to-text dictation and text-to-speech playback for hands-free interaction.

📎

Vision & Documents

Attach images and PDFs to any conversation and let the model analyze, summarize, or answer questions about them.

🤖

Agent Mode

Autonomous tool-calling loop — the model searches the web, saves memories, and chains actions to complete complex tasks.

🧠

Persistent Memory

Memory that persists across conversations — add your own notes or let the model build it automatically during chat.

GET STARTED

Up and running in two steps

  1. Deploy a LiteLLM proxy — or run Ollama locally
  2. Download OpenClient, open Settings, and enter your server URL
